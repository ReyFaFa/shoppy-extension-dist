🛍️ Shoppy Extension v1.6.6

Chrome 확장프로그램 자동 업데이트 시스템을 지원하는 Shoppy Extension 배포 페이지입니다.

📦 현재 버전: 1.6.6

최신 버전의 중국어-한국어 번역기 및 이미지 일괄 번역 기능을 제공합니다.

📥 다운로드

권장: ZIP 파일을 다운로드하여 개발자 모드로 설치하세요.

📦 ZIP 파일 다운로드 (개발자 모드용)
🔧 CRX 파일 다운로드 (고급 사용자용)

🚀 설치 방법

  1. ZIP 파일 다운로드 - 위 링크에서 ZIP 파일을 다운로드합니다
  2. 압축 해제 - 다운로드한 파일을 원하는 폴더에 압축 해제합니다
  3. Chrome 확장프로그램 페이지 - Chrome에서 chrome://extensions/로 이동합니다
  4. 개발자 모드 활성화 - 우측 상단의 "개발자 모드" 토글을 켭니다
  5. 확장프로그램 로드 - "압축해제된 확장 프로그램을 로드합니다" 클릭 후 폴더 선택

🔄 자동 업데이트

설치 후 Chrome이 자동으로 업데이트를 확인하고 적용합니다. 사용자의 별도 조치가 필요하지 않습니다.

업데이트 URL: https://reyfafa.github.io/shoppy-extension-dist/update.xml
